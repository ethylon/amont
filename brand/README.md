# Amont brand kit

The canonical source for Amont's visual identity. Open
[`guidelines/amont-brand-guidelines.html`](guidelines/amont-brand-guidelines.html) in a
browser for the full charte graphique (mark, palette, typography, voice).

The mark is a bird turned upward — the gaze toward the source, the lift of the current —
filled with the brand's single gradient, **Givre** (`#8FE3E8`) at the head down to **Abysse**
(`#0E2748`) at the foot. On light backgrounds the head darkens to `#3FB4C4` to keep its
contrast.

## Files

| Path | Use |
| --- | --- |
| `logo/amont-mark.svg` | Mark on dark backgrounds. Vector reference. |
| `logo/amont-mark-light-bg.svg` | Mark for light backgrounds (darkened head). |
| `logo/amont-mark-mono-white.svg` · `-mono-ink.svg` | Monochrome, constrained or print contexts. |
| `logo/amont-lockup-horizontal-dark.svg` · `-light.svg` | Default lockup (README, site, docs). |
| `logo/amont-lockup-stacked-dark.svg` | Splash, about, square formats. |
| `app/amont-appicon.svg` + `icon-*.png` | Application icon, 512 → 16 px. |
| `app/amont-appicon-dense.svg` | Tight-margin variant, source of the 16/32 px rasters. |
| `app/icon.ico` | Windows multi-resolution icon (256 → 16). |
| `web/favicon.ico` | Favicon (48/32/16). |
| `web/amont-og.svg` · `.png` | Social / OpenGraph banner, 1280 × 640. |

## Where these land in the repo

- `resources/icon.png` / `resources/icon.ico` — Electron app + installer icon
  (`resources/icon.png` is also regenerated by `scripts/make-icon.mjs`).
- `docs/logo.png` — README header logo.
- `site/public/icon.png` / `site/public/favicon.ico` — website favicon and touch icon.

The in-app mark is drawn inline (same path data) in
`src/renderer/src/components/ui/mark.tsx`, `scripts/make-icon.mjs`,
`src/renderer/index.html`, and `src/renderer/crash.html` — keep those four in sync.

The social banner (`web/amont-og.png`) is uploaded in the GitHub repository settings
(Social preview); it is not referenced by the build.
